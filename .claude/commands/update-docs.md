# ドキュメントの更新

信頼できるソースからドキュメントを同期します:

1. package.json の scripts セクションを読み込む
   - スクリプトリファレンステーブルを生成
   - コメントからの説明を含める

2. .env.example を読み込む
   - すべての環境変数を抽出
   - 目的と形式をドキュメント化

3. docs/CONTRIB.md を生成:
   - 開発ワークフロー
   - 利用可能なスクリプト
   - 環境構築手順
   - テスト手順

4. docs/RUNBOOK.md を生成:
   - デプロイ手順
   - 監視とアラート
   - よくある問題と解決策
   - ロールバック手順

5. 古くなったドキュメントを特定:
   - 90日以上更新されていないドキュメントを検索
   - 手動レビュー用にリストアップ

6. 差分サマリーを表示

信頼できるソース: package.json と .env.example
