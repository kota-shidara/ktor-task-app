# ビルドとエラー修正

TypeScriptおよびビルドエラーを段階的に修正する:

1. ビルド実行: npm run build または pnpm build

2. エラー出力を解析:
   - ファイルごとにグループ化
   - 重要度順にソート

3. 各エラーに対して:
   - エラーの前後5行のコンテキストを表示
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - ビルドを再実行
   - エラーが解消されたことを確認

4. 以下の場合は停止:
   - 修正が新たなエラーを引き起こした場合
   - 3回試行しても同じエラーが解消しない場合
   - ユーザーが一時停止を要求した場合

5. サマリーを表示:
   - 修正済みエラー数
   - 残存エラー数
   - 新たに発生したエラー数

安全のため、エラーは一つずつ修正すること!
