# Switch to Main

mainブランチに切り替えて最新の状態に更新します。

## 手順

以下の手順を順番に実行してください：

### 1. 現在の状態を確認
- `git status` で未コミットの変更がないか確認
- 未コミットの変更がある場合は、ユーザーに警告して続行するか確認

### 2. mainブランチに切り替え
- `git checkout main` でmainブランチに切り替え
- mainブランチが存在しない場合は `git checkout master` を試行

### 3. 最新を取得
- `git pull` でリモートから最新の変更を取得

### 4. 結果を報告
- 切り替え前のブランチ名
- pull結果（更新されたファイル数など）

## 注意事項
- 未コミットの変更がある場合は、stashするか確認すること
- エラーが発生した場合は適切に処理し、ユーザーに報告すること
